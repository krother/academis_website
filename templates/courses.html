Training Methodology

In the 21st century, the job of a trainer is not just providing information. You can get as much information as you like, anytime, without my help. A good trainer will show a path, inspire and help overcome obstacles. This is where 25 years of experience as a trainer and software developer are the most useful.

I believe the best setting for effective training is in-person. In my courses, I use a vast repertoire of methods to make training and teaching enjoyable and effective: recap quizzes, fixing buggy code, carefully thought programming quests and many more. You can find my teaching methods documented on the Teaching Programming page.

Since 2025, I am running a training room in Berlin-Schöneberg that fits up to 8 persons. You can book courses here. My weekly morning and evening classes take place there as well.

Advanced Python

In this course, you will write an object-oriented program in Python. You will gradually add automated tests and other tools to maintain a high quality of your code. This course empowers you to write and maintain larger programs in Python. To participate in the course, you should be be familiar with basic Python data structures and commands. You are able to execute standalone Python programs. You can move around in the command line and communicate with a git repository.

Course Topics

  • set up your Python project
  • design classes in Python
  • structure modules and packages
  • write Unit and Integration Tests
  • refactoring
  • type annotations
  • debugging

Data Analysis with polars

In this course, you will guide a crew of polar bears on a research expedition through arctic underwater worlds. Keep their vessel on course, solve icy data problems and you may eventually find traces of the fabled penguins.

The course introduces you to polars, a modern component of the Python data science stack. Use it to wrangle challenging data sets, create enticing plots and combine your results into a live dashboard. The course discusses typical data biases and statistical tools that everybody working with data should have in their toolbelt.

No previous programming knowledge is required.

Database Infrastructure

In this course, you learn to work with the most popular SQL and NoSQL databases. You will create instances of each database in Docker, fill them with data and write Python clients for each of them. You learn to know the strengths and weaknesses of each architecture in order to build reliable data infrastructure.

  • schemas in MongoDB
  • Postgres
  • analyzing performance
  • data ingestion
  • indexing text and geo data
  • DB authentication mechanisms
  • Docker deployment
  • Sharding and Replication

Machine Learning Fundamentals

This course covers the academic foundations of Machine Learning. It starts with the basics of Linear Regression – a method that often turns out to be not so basic already – and ends with Artificial Neural Networks up to LLMs.

The course places an emphasis on tools to train models consistently and their critical evaluation. It covers more evaluation metrics than actual models and goes through a variety of regularization techniques.

The entire course is powered by the standard Python stack including scikit-learn, statsmodels, TensorFlow and PyTorch. Basic knowledge of Python is highly recommended.
