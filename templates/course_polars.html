Data Analysis with polars

In this course, you will guide a crew of polar bears on a research expedition through arctic underwater worlds. Keep their vessel on course, solve icy data problems and you may eventually find traces of the fabled penguins.

Goals

You will learn to analyze a tabular dataset using the Python programming language. The course introduces you to polars, a modern component of the Python data science stack. You will wrangle challenging data sets, calculate statistical metrics, create enticing plots and combine your results into a live dashboard. The course discusses typical data biases and statistical tools that everybody working with data should have in their toolbelt.

The main goal of the course is to enable you to quickly summarize a dataset and communicate your findings clearly.

How the course works

The course consists of interactive lectures. During the lectures, we will discuss key concepts from the field of data science and run excample code using Python, polars and seaborn. There will be good and weak examples of data analyses so that you learn to critically observe data presented to you.

The course material contains plenty of exercises that participants are expected to work on their own to build practical data processing skills in Python/polars. During classes, they get support with technical questions. participants are encouraged to bring their own questions and bugs found during the practical exercises. The course materials are permanently available on under a Creative Commons 4.0 License.

Prerequisites

To participate in the module, fluency in Python is useful but not strictly required. You should be able to write simple instructions and debug smaller pieces of Python code. To work on the exercises and try out the examples, you need to be able to execute code in a Jupyter Notebook. Writing clean code and keeping Jupyter Notebooks tidy matters and is more important than knowing lots of Python.

There is little need for structured programming, loops, functions or classes when exploring data. No programming outside Jupyter is required for this module. No knowledge on advanced statistics, databases or Machine Learning is required for this course.
