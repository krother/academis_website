Advanced Python

How do I develop a large program? This course takes you past the language syntax towards the deeper questions in programming. While writing an object-oriented software in Python, you will make many decisions regarding the project setup, structure and composition of classes. I would like to show you time-tested patterns to make these decisions in a robust way.

Goals

During the course, you will develop a Python program from scratch. You start by setting up a clean project structure. When the code becomes larger, you will decompose it into functions and modules. When it becomes even larger, you will also need classes. To neep the code working as you are making changes, you will also need an automated test suite and a few static code checkers. The course discusses typical challenges in a software project and strategies to overcome them.

The main goal of the course is to enable you to write well-structured Python software.

How the course works

The course consists of a smaller theoretical part and plenty of exerrcises. During the exercises, you will implement a Python application step by step, incorporating key concepts from the theoretical parts. There will be some extra problems that come your way that you need to solve as a team.

The course material contains background reading and code examples that you can use to deepen your knowledge after the course. During classes, you get support with technical questions. We will also conduct some code reviews together. The course materials are permanently available on under a Creative Commons 4.0 License.

Prerequisites

To participate in the course, you should be be familiar with basic Python data structures and commands. You should be able to execute standalone Python programs and communicate with a git repository. This course is also suited for people who have mostly worked in Jupyter notebooks.
